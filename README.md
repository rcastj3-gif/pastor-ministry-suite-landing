# Pastor Ministry Suite Landing Page

A high-converting landing page for the Pastor Ministry Suite product.

## Features

- Hero section with strong headline and CTAs
- 7 agents showcase with time savings
- Cost comparison and savings calculator
- Testimonials from pastors
- 3-tier pricing section
- FAQ accordion
- Final CTA section

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.

## Build for Production

```bash
npm run build
```

The static export will be in the `dist` folder.

## Deploy to Vercel

1. Push this repo to GitHub
2. Connect Vercel to your GitHub repo
3. Deploy!

## Environment Variables

None required for the landing page.
